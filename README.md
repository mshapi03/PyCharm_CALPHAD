# Learning and Implementing the pycalphad Package
_Author: Mitchell Shapiro-Albert_ 

_Date Started: April 17, 2026_

This repository contains python scripts designed to familiarize users with the pychalphad package, a Python library for 
computation thermodynamics using the CALPHAD method. Official information from the developer team can be found here: 
https://pycalphad.org/docs/latest/index.html. The author and contributors to this repository do not claim ownership of
this packages nor have contributed to it. We are simply using this for graduate coursework and for educational purposes. 

### File Structure

_requirements.txt_ or _calphad_practice.toml_ contain the required imports to run the main script and generate figures.
_main.py_ is the working script which contains novel function definitions and codes which read in thermodynamic database
files (.tdb) and create figures. The _TDB_Files_ folder contains a few freely available .tdb files to play around with
and troubleshoot issues with the _main.py_ script.
